# Plan Finanzas Personal

A full-stack personal finance application built with React, NestJS, and PostgreSQL.

## Features

- **Dashboard**: Overview of finances.
- **Transactions**: Track income and expenses.
- **Accounts**: Manage multiple bank accounts/wallets.
- **Budgeting**: Set limits and goals.
- **Reports**: Visual analytics.

## Tech Stack

- **Frontend**: React (Vite), TypeScript, Tailwind CSS, Recharts.
- **Backend**: NestJS, TypeORM, PostgreSQL.

## Getting Started

1.  **Database**: Ensure PostgreSQL is running and `.env` in `server/` is configured.
2.  **Backend**:
    ```bash
    cd server
    npm install
    npm run start:dev
    ```
3.  **Frontend**:
    ```bash
    cd client
    npm install
    npm run dev
    ```
